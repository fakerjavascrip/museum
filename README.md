# Museum Exhibit — *There There*

Interactive static exhibit: **Reconstructing Urban Indian Identity Through Technology** (Tommy Orange, *There There*).

## Live site

After deployment: https://fakerjavascrip.github.io/museum/

## Local preview

Open `index.html` in a browser, or run a local server:

```bash
python -m http.server 8080
```

Then visit http://localhost:8080

## Structure

- `index.html` — main page
- `css/` — styles
- `js/` — exhibit data and interactions
- `img/` — artifact images
